# 🕊️ Peace Palace Ticket Watcher

This project checks for new **Peace Palace guided tour tickets** via the TicketTeam API.

It supports:

- 🧠 Filters for "Inside the Palace" and/or Garden tours
- 🇬🇧🇳🇱 Dutch and English options
- 📬 Telegram alerts:
  - Emergency alerts when new tickets appear
  - Daily status report of filtered tours
- 🕰️ Scheduled runs using GitHub Actions
- 🖥️ Local testing with `.env`
